### Overview

JS Library of common data structures inspired by Cracking the Code

### Get Started

```bash
#Install
npm i dsjs

#Watch for changes
npm run test:watch

#Run tests
npm t

#Run individual tests
npm t -- <name-of-test-file>

#Compile for npm publication
npm run build
```
